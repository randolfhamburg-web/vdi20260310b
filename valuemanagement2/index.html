Funktionskostenanalyse
Methodik: Value Index = Relativer Funktionswert (%) / Relativer Funktionskostenanteil (%). Wert > 1,0 → Unterbewertet (Kostenpotenzial erschöpft). Wert < 1,0 → Optimierungsbedarf. Wichtigkeiten werden normiert (Summe = 100%). Haupt- und Nebenfunktionen werden separat ausgewiesen.
